## 冒泡排序

---

冒泡排序的思想：两两比较相邻记录的关键码，如果反序则交换，直到没有反序的记录为止。

（1）将整个待排序的记录序列划分成有序和无需区，初始状态下有序区为空，无序区包含所有的待排序记录。

（2）对无序区从前向后依次将相邻记录关键码进行比较，若反序则交换，从而使得关键码小的记录向前移动，关键码大的向后移动。

（3）重复执行（2），直到无序区中没有反序的记录。

```
public void bubbleSort(int[] a){
    int i = a.length-1;
    while(i>0){
        int pos = 0;
        for(int j =0;j<i;j++){
            if(a[j]>a[j+1]){
                pos = j;
                int temp = a[j];
                a[j] = a[j+!];
                a[j+1] = temp;
            }
        }
    }
}
```
